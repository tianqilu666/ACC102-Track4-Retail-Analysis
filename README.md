# ACC102 Track4 - 2025 Global Retail Sales Interactive Tool
## 1. Problem & Target User
This interactive tool helps business students and analysts quickly visualize and interpret 2025 global retail sales performance. The target users include accounting learners, finance analysts and retail industry beginners.

## 2. Data Information
- Data Type: Simulated monthly retail sales data
- Access Date: 2026-04-23
- Key Variables: Month, Walmart, Carrefour, Costco, Tesco, Aldi
- Data Unit: Billion US dollars

## 3. Technical Methods
Python libraries: streamlit, pandas, numpy, matplotlib
Workflow: Data generation → Data sorting → Metric calculation → Visualization → Insight output

## 4. Key Findings
1. Walmart ranks first in total annual retail sales
2. All selected retail brands maintain continuous growth in 2025
3. Aldi shows the most obvious upward growth trend

## 5. Local Execution Guide
1. Clone or download this repository
2. Install dependencies: pip install -r requirements.txt
3. Launch the application: streamlit run app.py
